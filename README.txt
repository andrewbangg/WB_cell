1. Установить Python
1.1 В командной строке перейти в папку с программой для печати ячеек и ввести команду: python -m venv venv
2. В файле test_bot.bat изменить путь, на тот который у пользователя (где лежит программа для печати)
3. В файле copy_db_wb.py изменить пути:
	Певрый путь где лежит БД WB
	Второй путь где лежит программа для печати ячеек
ВСЕ ИЗМЕНЕНИЯ СОХРАНЯЙ
4. Открыть командную строку, перейти в папку где лежит программа для ячеек и выполнить команды:
	4.1 cd venv\Scripts
	4.2 activate
	4.3 cd..
	4.4 cd..
	4.5 pip install -r requirements.txt
Дожадться установки, закрыть командную строку
5. Попробовать запустить BAT file -- test_bot.bat
